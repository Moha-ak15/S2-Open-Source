# S2-Open-Source
## NPM Installation 
```bash
npm install
```

## jest Installation
#### this command create a basic package.json file with default values
```bash
npm init -y
```

#### This will add Jest to the devDependencies section of your package json file.
```bash
npm install --save-dev jest
```
